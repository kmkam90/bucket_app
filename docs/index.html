개인정보 처리방침

최종 수정일: 2026년 3월 16일

버킷 앱(이하 "본 앱")은 사용자의 개인정보를 소중히 여기며, 아래와 같이 개인정보 처리방침을 안내합니다.

1. 수집하는 개인정보

본 앱은 사용자의 개인정보를 수집하지 않습니다.

  • 회원가입이나 로그인을 요구하지 않습니다.
  • 이름, 이메일, 전화번호 등 개인 식별 정보를 수집하지 않습니다.
  • 위치 정보를 수집하지 않습니다.

2. 데이터 저장

사용자가 앱 내에서 생성하는 모든 데이터(버킷리스트, 목표, 진행률 등)는 사용자의 기기에만 로컬로 저장됩니다.

  • 외부 서버로 데이터를 전송하지 않습니다.
  • 제3자와 데이터를 공유하지 않습니다.
  • 앱을 삭제하면 모든 데이터가 기기에서 삭제됩니다.

3. 분석 및 광고

  • 사용자 행동 분석 도구를 사용하지 않습니다.
  • 광고를 표시하지 않습니다.
  • 광고 식별자를 수집하지 않습니다.

4. 아동 개인정보

본 앱은 아동을 대상으로 하지 않으며, 의도적으로 13세 미만 아동의 개인정보를 수집하지 않습니다.

5. 변경 사항

본 개인정보 처리방침은 변경될 수 있으며, 변경 시 앱 업데이트를 통해 고지합니다.
